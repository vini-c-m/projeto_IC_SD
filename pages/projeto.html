A Caixa Chata - Projeto Detalhado

Tema e Objetivo

O projeto "A Caixa Chata" tem como objetivo desenvolver um sistema automatizado utilizando Arduino que demonstra conceitos fundamentais de sistemas digitais e programação. O sistema foi projetado para resolver problemas práticos através de automação inteligente.

Componentes Utilizados

  • Arduino Uno (microcontrolador principal)
  • Sensores diversos (temperatura, movimento, luminosidade)
  • LEDs e resistores
  • Servo motor
  • Módulo WiFi
  • Display LCD
  • Botões e switches
  • Protoboard e jumpers

Código Arduino


// Código principal do projeto A Caixa Chata
#include <Servo.h>
#include <LiquidCrystal.h>

// Definição de pinos e variáveis
const int sensorPin = A0;
const int ledPin = 13;
const int servoPin = 9;

Servo meuServo;
LiquidCrystal lcd(12, 11, 5, 4, 3, 2);

void setup() {
  // Inicialização do sistema
  Serial.begin(9600);
  pinMode(ledPin, OUTPUT);
  meuServo.attach(servoPin);
  lcd.begin(16, 2);
  lcd.print("Sistema Ativo");
}

void loop() {
  // Loop principal do programa
  int sensorValue = analogRead(sensorPin);
  
  if (sensorValue > 500) {
    digitalWrite(ledPin, HIGH);
    meuServo.write(90);
  } else {
    digitalWrite(ledPin, LOW);
    meuServo.write(0);
  }
  
  delay(100);
}
